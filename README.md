O **Sistema de Estudo e Análise de Coloração de Pele em Humanos Virtuais** é um software voltado para o estudo do comportamento dos tons de pele em humanos virtuais, e como os algoritmos se comportam em diferentes variantes e ambientes de estudo, além de como diferentes algoritmos podem ter diferentes níveis de performance dentro do nosso sistema e com nossos objetos de estudo. As areas de estudo que englobam o uso desse algoritmo são:

- Processamento de Computação Gráfica
- Inteligência Artificial
- Vision Model Language (VML)
- Humanos Virtuais
